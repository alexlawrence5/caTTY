# CaTTY, what is it?🤔
CaTTY is an collection of cool apps such as Cmatrix, fastfetch etc. It has his own independent basic interactive shell too! (CaSH)

# Platform support💻
Manjaro, EndevaourOS, Arch, Arco, Nyarch, etc. (All Arch-based distributions who use pacman package manager)

# Why not other distributions such as Debian?
To be honest, I don't know. I think maybe because most people use this tools on Arch Linux, but I will try to add other distribution support too.
